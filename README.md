springwork，在spring中整合关键技术。

目前已有： （后面慢慢加）
1.springwork-common
	仅作为通用类存储项目
	
2.springwork-model
	实体类项目
	
3.springwork-security
	oauth2单点登录及授权码模式等
	
4.springwork-mybatis
	整合mybatis
	
5.springwork-web
	web项目，简单提供了个页面。
	
6.springwork-dubbo-produce
	dubbo提供方
	
7.springwork-dubbo-consume
	dubbo-consume
	
8.springwork-activemq
	activemq接收处理项目
	
9.springwork-netty
	netty服务器，配置启动，责任链handler。

10.springwork-hibernate
    一个简单的hibernate demo
